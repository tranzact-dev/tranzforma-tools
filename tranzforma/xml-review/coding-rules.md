# fusionplace フォーム コーディングルール

> Uniforma（xml-review ツール）が自動チェックする項目に基づく、フォーム定義のルールです。
> 
> あくまで保守性・可読性維持、潜在的な不具合防止のためのルールです。
> 調査や確認、移行用のフォームなど、一時的に用いるフォームにまで適用する必要はありません。
> 目的を考え、バランス感覚を持って運用しましょう。

---

## 式の書き方

フォームのタイトル式・数式・メンバー条件・ドリルダウン条件・ローカルマスクなどに共通するルールです。

### 小文字で書くもの

| 対象 | NG | OK | 自動修正 |
|---|---|---|---|
| **メソッド**（`.`の後） | `!ACCOUNT.Label` | `!ACCOUNT.label` | ✅ |
| **予約語** | `IF ~ THEN ~ ELSE ~ ENDIF` | `if ~ then ~ else ~ endif` | ✅ |
| **関数**（`@`の後） | `@TEXT(...)` `@ROUND(...)` | `@text(...)` `@round(...)` | ✅ |

> 予約語一覧: `if` / `then` / `else` / `endif` / `and` / `or` / `not` / `true` / `false` / `dimensions` / `ledgers` / `editions`

### 大文字で書くもの

| 対象 | NG | OK | 自動修正 |
|---|---|---|---|
| **ラベル**（`!`の後） | `!#account` `!cat` | `!#ACCOUNT` `!CAT` | ✅ |
| **疑似メンバー** | `@cur.label` `@pov` | `@CUR.label` `@POV` | ✅ |

> 疑似メンバー: `@CUR` / `@POV` / `@RKEY` — ドット以降は小文字のまま（`@CUR.label`）

### #LEAF の書き方（ERROR）

`#LEAF` の値は必ず **ダブルクォーテーション付き・大文字** で記述してください。自動修正されません。

```
✅  #LEAF = "TRUE"    #LEAF = "FALSE"
❌  #LEAF = TRUE      #LEAF = "true"    #LEAF = "True"
```

## フォーム設計

### フォームラベルにハイフンを使わない

ハイフンはソート順のずれやスクリプトの誤動作の原因になります。アンダースコアを使ってください。

```
❌  SALES-REPORT       →  ✅  SALES_REPORT
```

### 行仕様/列仕様のラベルを数字だけにしない（ERROR）

数字のみのラベルはオフセット参照と解釈され、数式が誤動作します（ver.15以降）。

```
❌  1    100           →  ✅  ROW01    PREV_SALES
```

### デフォルトタイトルを残さない

`Column Title` / `Row Title`（`列タイトル` / `行タイトル`）は必ず適切な内容に書き換えてください。

---

## 自動修正について

上の表で「✅」がついた5項目（メソッド・ラベル・予約語・関数・疑似メンバーの大文字小文字）は、Uniforma のチェック後にワンクリックで自動修正できます。

それ以外（`#LEAF`・ハイフン・数字ラベル・デフォルトタイトル）は手動修正が必要です。

---

## その他のチェック・情報（INFO / 参考）

Uniforma は上記のルールチェックに加えて、以下のような設計上の注意点も検出・表示します。
意図的な設定であれば問題ありませんが、設定ミスの早期発見に役立ちます。

- **起点メンバと展開方法の想定外の組み合わせ** — `ALL` / `ROOTS` に `NONE_EXPANSION` 以外を指定していないか等
- **トリガー設定の有無** — 意図しない自動更新が発生していないかの確認
- **エクスポート時の行ループ出力** — エクスポート有効なのに行ループ出力が無効になっていないか
- **セル保護の有無** — `protected = true` の検出
- **行ループのインデント幅** — ループ側のインデントがレポート書式のインデントより小さいと効果がないので通知します
- **ラベル・名前が両方空欄** — スペーサー行かどうかの確認
- **元帳マスク設定** — ローカルマスクが設定されている場合の通知
- **参照フォーム固有** — タイトル未設定、ループ項目選択行の表示、ループ表示設定、ドリルダウン条件の空欄
- **インポートフォーム固有** — インポート仕様の有効性、符号表示タイプ、ヘッダースキップ行数
- **エクスポートフォーム固有** — エクスポート仕様の有効性、列/行ヘッダーの抑制
- **パイプライン出力固有** — 非末端ループのタイトルに `.label` が含まれているか
